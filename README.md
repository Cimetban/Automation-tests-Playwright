# ShopEase

A simple shopping website project built with HTML, CSS, and JavaScript.

## Features

- Product grid with featured items
- Login page with dummy authentication
- Payment form for checkout
- Cart quantity controls and remove actions
- Live cart total calculation
- Responsive layout for mobile and desktop

## Getting started

1. Open `index.html` in your browser
2. Browse the products and add items to the cart
3. Use the cart panel to adjust quantities or proceed to checkout
4. Visit `login.html` to sign in, then `payment.html` to complete payment

## Customize

- Edit `products` in `script.js` to change store items
- Update `styles.css` to adjust the visual theme
- Add product categories or search functionality as needed
